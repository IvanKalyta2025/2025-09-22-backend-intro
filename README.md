## Command shortlist (The most important)

- Get Help

  ```sh
  dotnet --help
  ```

-- Get Version

  ```sh
  dotnet --version
  ```

- Generate new project

  ```sh
  dotnet new sln
  dotnet new web
  dotnet sln add <name-of-.csproj>
  ```

- Start the project

  ```sh
  dotnet run
  ```

- Stop the server (any console program really)

  ```sh
  Ctrl + C
  ```
